# Book Folding Pattern Generator

This is a single-page application that allows users to generate book folding patterns from an image.

## Development

To run the project locally:

1.  Navigate to the project directory: `cd book-folding-pattern-generator`
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
    The application will be available at `http://localhost:5173` (or another port if 5173 is in use).

## Build and Deploy

To build the project for deployment as a static application:

1.  Navigate to the project directory: `cd book-folding-pattern-generator`
2.  Build the project: `npm run build`
    The build artifacts will be generated in the `dist/` directory. These files can be served by any static file server.
